# NumPy Data Types and Type Conversion

This project demonstrates how to work with different NumPy data types in Python and how data types affect memory usage and performance.

## Topics Covered

* Integer data types (`int8`, `int16`, `int32`, `int64`)
* Floating-point data types (`float16`, `float32`, `float64`)
* Boolean data type (`bool_`)
* String data type (`str_`)
* Object data type (`object_`)
* Checking array data types using `dtype`
* Measuring memory usage using `nbytes`
* Converting arrays from one data type to another using `astype()`

## Examples Included

* Creating NumPy arrays with custom data types
* Comparing memory consumption of different data types
* Converting integer arrays to floating-point arrays
* Converting floating-point arrays to integer arrays
* Understanding how NumPy handles strings, booleans, and objects

## Learning Objectives

By completing this project, you will learn:

* How NumPy stores data efficiently
* Why choosing the correct data type matters
* How to reduce memory usage with appropriate data types
* How to perform data type conversion in NumPy

## Technologies Used

* Python
* NumPy
